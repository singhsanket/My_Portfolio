My projects

Bank Management Model

The Bank Management System is a Java-based desktop application that helps manage a bank’s operations efficiently. The project is built using Java Swing for the graphical user interface (GUI), JDBC (Java Database Connectivity) for database connectivity, and MySQL as the backend database to store account details, transactions, and other necessary data. This system simulates real-life banking operations, allowing users to manage accounts, perform transactions, and track their account balances.

Heart Disease Prediction Model

A Heart Disease Prediction Model is a machine learning model designed to predict the likelihood of an individual developing heart disease based on various factors and medical data. The model uses historical data of patients, including their health attributes, to classify whether they are at high risk of heart disease.
